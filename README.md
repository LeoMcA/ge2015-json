# ge2015-json

Generate a JSON file of constituencies, candidates and results of the 2015 General Election in the UK.

## Usage

```sh
git clone --recursive https://github.com/LeoMcA/ge2015-json.git
cd ge2015-json
npm install
node generate.js > output.json
```
